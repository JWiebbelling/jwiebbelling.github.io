Empanada

A simple 2D platformer made in Unity.

Empanada is a very simple 2D platformer made in Unity. The goal of this project was to learn certain features of 2D games and implement them in the game itself. Working on it was quite fun! It was the second platformer I've made, and I learned quite a lot from it.
I'm overall satisfied with the results. The major concept of this project is that you're an astronaut that needs to get out of space as soon as possible. You do that by finishing every level without dying. Killing enemies is optional!
